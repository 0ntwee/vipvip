## ⚠️ Критические моменты из гайда

- **FILESTREAM при установке SQL:** все 3 галки должны быть установлены! После установки изменить нельзя
- **Порт SQL:** обязательно 49675
- **Пароль SQL:** `123qweR%` (запомни!)
- **Лицензия ЦУС:** должна заканчиваться на `-1` (если 26022 — не та)
- **Агрегирование:** настраивай ТОЛЬКО интерфейс, смотрящий в "интернет" координаторов. Трогаешь локальный — отвалится веб-интерфейс
- **Ключи координаторов:** делай образы ISO через UltraISO, подключай как CD
- **Отправка ключей при компрометации:** только на клиента, иначе не дойдёт

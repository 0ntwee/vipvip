# Сетевая конфигурация

## Сети VMware
1. **Host-only/Внутренняя** - Сеть центрального офиса (ЦО)
2. **Host-only/Внутренняя** - Сеть филиала
3. **Host-only/Внутренняя** - Сеть межсетевого взаимодействия
4. **Host-only/NAT/Bridge** - Виртуальный "Интернет"

- **[3:00]** Аккредитованный режим настроен, компрометация выполнена

##  Если застреваешь

- **SQL не ставится** → проверь FILESTREAM (3 галки), перезапусти установку
- **ЦУС не подключается к БД** → проверь IP БД, имя БД, пароль sa=`123qweR%`, порт 49675
- **Координатор не видит клиентов** → проверь образ диска с ключами подключен как CD
- **Ключи не доходят** → отправь только на конкретного клиента, не на всех
- **Веб-интерфейс координатора отвалился** → тронул не тот интерфейс при агрегировании, исправляй через консоль координатора
